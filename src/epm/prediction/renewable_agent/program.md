# 新能源出力预测 Agent — program.md (optional, v0.1)

## 研究方向

超短期 15 分钟功率预测，归一化至装机容量。

可修改范围：数值天气预报后处理、空间插值、云团追踪特征、历史相似日匹配。

## 约束

主指标：归一化 RMSE（与负荷不同 — 这里的偏差不直接触发罚款，
所以可以用 RMSE）。

## 评估协议

1. 归一化 RMSE
2. 极端天气日子集（雷雨、强风等）
3. 推理时间

仅当 `MarketRules.name` 对应省份新能源装机占比 > 25% 时启用此 Agent；
否则把新能源特征合并进 `price_agent`.
